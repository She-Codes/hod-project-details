House Minutes Application

Virginia House of Delegates application used to record the journal day of a legislative session.

This app was built using a Rails backend and a combination of Bootstrap, jQuery, vanilla JavaScript, and SASS for the frontend. The majority of the JavaScript was written by myself using vanilla when possible, with some jQuery sprinkled in to work with Bootstrap plugins and for AJAX calls to the backend APIs. The app is used in real-time during a legislative session so speed, accuracy and ease-of-use is paramount.

Intro Screen

This was just a fun intro I added to the app to greet the user once they login. This was built entirely in CSS and uses CSS animations. For a live example, you can take a look at this Codepen

App Menus

There are several areas of the app that require their own menus. The user is easily able to transition between the main application menu and the current menu unique to that area of the app. There is also the ability to hide all menus and expand the main viewing area.

Menu Actions

This is an example of communication with the backend API. A user is able to select multiple rows and apply a menu action, the results of which are immediately reflected in the corresponding rows via AJAX. There are several other features in this section of the app including disabling menus depending on which selections are made, deleting previously added actions, and adding custom actions not available by menu.

Custom Actions

As mentioned above, the user can dynamically add a custom action without using the menu. A small CSS animation provides feedback to the user that the action was saved on the server side.

Detecting Changes

Once the document is changed, the indicator on the save button changes to alert the user that changes need to be saved. The app detects changes on keyup, whether items have been rearranged via 'drag and drop', and whether items have been added via the menus. If the user tries to navigate away without saving changes, they receive a warning.

Deleting Items

Items can easily be deleted using a control that appears when the item item is either hovered or active. Because this is a destructive action the UI prompts the user to click twice by initially changing shape. To cancel a delete, the user can simply click elsewhere on the page.
